# 🥐 Lumière Bakery

Um aplicativo mobile moderno para uma padaria artesanal de alto padrão, focado em experiência do usuário (UX) e design sofisticado.

![Expo SDK](https://img.shields.io/badge/expo-v55-black?logo=expo)

---

## 🎨 Sobre o Projeto

O app **Lumière Bakery** foi desenvolvido para oferecer uma navegação fluida entre o cardápio de pães artesanais e o sistema de reserva de mesas. O design utiliza uma paleta de cores creme (`#FDF8F2`) e preto, transmitindo elegância e conforto.

---

## 🚀 Funcionalidades

* **Home:** Navegação rápida e ações principais com visual limpo.
* **Cardápio (Menu):** Filtragem por categorias (Breads, Pastries, etc.) com troca de estados dinâmica.
* **Reservas (Book Table):** Interface intuitiva com calendário e seleção de horários.
* **Navegação Nativa:** Implementada com **Expo Router** para transições fluidas entre telas.

---

## 🛠️ Tecnologias Utilizadas

* **React Native** com **Expo SDK 55**
* **Expo Router** (Navegação baseada em arquivos)
* **StyleSheet** (Estilização customizada)

## ⚙️ Como Rodar o Projeto

### 1. Clone o repositório

```bash
git clone https://github.com/GhostDev-Creator/LumiereBakery
```

### 2. Entre na pasta do projeto

```bash
cd seu-repositorio
```

### 3. Instale as dependências

```bash
npm install
```

### 4. Inicie o Expo

```bash
npx expo start
```
